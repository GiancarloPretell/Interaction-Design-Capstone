# Interaction Design Capstone

## Overview
This project is a redesign of the Yale School of Art website based on our Figma prototype. The goal of this project is to recreate the website using HTML, CSS, and JavaScript while improving usability, layout clarity, and overall user experience.

The site is built with a mobile-first approach and adapts to both mobile and desktop screen sizes.

## Features
- Responsive layout for mobile (~375px) and desktop (~1280px)
- Mobile navigation with a working hamburger menu
- Desktop navigation with a full horizontal menu
- Multiple pages connected with working links
- Layout and design closely matching the Figma prototype

## How to Run Locally
1. Download or clone the repository
2. Open the project folder
3. Double click `index.html`  
   OR right click → open with browser (Chrome recommended)

No installation or dependencies are required.
